# 장바구니 페이지 리뷰

## 고객사 요구사항(요약)
- 2주 내 제작, 중간 점검 1회
- 여러 상품을 한 번에 비교·선택 가능해야 함
- 금액 표시 정확성 최우선
  - 단가 × 수량 = 소계
  - 총 상품 금액 + 배송비 = 결제 예정 금액
- 10만 원 이상 구매 시 배송비 무료(화면에 잘 보이게, 계산에도 반영)
- 테스트 주소 제공 → 수시 확인 및 피드백
- 디자인 시안 우선(버튼 위치, 글자 크기·색 등)
